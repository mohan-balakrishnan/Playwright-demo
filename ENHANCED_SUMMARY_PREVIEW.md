## Status Indicators Explained

### 🟢 ALL PASSED
- All tests passed successfully
- No failures or errors
- 100% pass rate

### 🟡 SOME FAILED
- Some tests failed
- More tests passed than failed
- Pass rate between 1% and 99%

### 🔴 MOST FAILED
- Most tests failed
- More tests failed than passed
- Pass rate below 50%

## How It Works

The workflow:
1. Runs Playwright tests
2. Generates `test-results/results.json`
3. Parses JSON using `jq` (JSON processor)
4. Calculates statistics
5. Formats output as Markdown
6. Writes to `$GITHUB_STEP_SUMMARY`
7. GitHub displays it in the Summary tab

---

## Technical Details

### Data Source
- **File:** `test-results/results.json`
- **Format:** JSON
- **Generated by:** Playwright JSON reporter

### Parsing Tools
- **jq:** JSON query processor
- **awk:** Text processing
- **bc:** Calculator for percentages

### Output Format
- **Markdown:** GitHub-flavored markdown
- **Location:** `$GITHUB_STEP_SUMMARY` environment variable
- **Display:** GitHub Actions Summary tab
